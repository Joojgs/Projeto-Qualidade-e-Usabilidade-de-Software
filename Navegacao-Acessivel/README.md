# Aplicativo de Navegação Acessível

Protótipo de aplicativo de navegação voltado para pessoas com deficiência visual.

## Sobre o projeto

O objetivo é criar uma solução acessível utilizando:

- comandos de voz
- instruções em áudio
- feedback tátil
- interface com alto contraste

## Tecnologias

- Python
- Kivy

## Estrutura

- app/
- screens/
- tests/

## Como executar

1. Clone o projeto:

    git clone LINK_DO_REPOSITORIO


2. Entre na pasta:

    cd Navegacao-Acessivel

    cd Prototipo-Inicial


3. Crie ambiente virtual:

    python -m venv venv


4. Ative:

    Windows:
    venv\Scripts\activate


5. Instale dependências:

    pip install -r requirements.txt


6. Execute:

    python -m app.main

    ou

    python -m Prototipo-Inicial.app.main
