# Aplicativo de Navegação Acessível

Protótipo de aplicativo de navegação voltado para pessoas com deficiência visual.

## Sobre o projeto

O objetivo é criar uma solução acessível utilizando:

- comandos de voz
- instruções em áudio
- feedback tátil
- interface com alto contraste

## Tecnologias

- Python
- Kivy

## Estrutura

app/
screens/
tests/

## Como executar

Clone o projeto:

git clone LINK_DO_REPOSITORIO


Entre na pasta:

cd Navegacao-Acessivel

cd Prototipo-Inicial


Crie ambiente virtual:

python -m venv venv


Ative:

Windows:
venv\Scripts\activate


Instale dependências:

pip install -r requirements.txt


Execute:

python -m app.main
